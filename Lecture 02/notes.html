📒 Notes for Lecture 02: Basic Web Page Setup

  • Doctype: Start with <!DOCTYPE html> to declare HTML5.
  • <html> tag: It wraps the full HTML document. Use lang="en" to set the language.
  • <head> section: Contains metadata like:
    • <meta charset="UTF-8"> – supports Hindi and emojis.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0"> – makes it responsive.
    • <title> – controls the browser tab name.
    • <link rel="stylesheet" href="style.css"> – links the CSS file.
  • <body> tag: Holds the visible content like text or buttons.
  • JavaScript: Use <script src="script.js"></script> to add a JavaScript file.
  • Content written: Inside body: "Hey this is my first website..." etc.

Hinglish: Is lecture mein humne apna pehla basic HTML page banaya. Starting hoti hai <!DOCTYPE html> se, jisse browser ko pata chalta hai ki HTML5 use kar rahe hain. Head mein metadata hota hai jaise charset, viewport, title. CSS file <link> se jodte hain. JavaScript ke liye <script> tag use hota hai. Body mein likha gaya text hi browser mein dikhai deta hai.
