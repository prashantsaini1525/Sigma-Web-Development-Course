  • <img> Tag: Display an image with:
    • src="Squid Game.png" ‚Äì source file
    • alt="Squid" ‚Äì alternate text for errors
    • height="200" ‚Äì sets height; width auto-adjusts
  • Table Structure:
    • <table> wraps the entire table
    • <caption> gives a title (‚ÄúEmployee Details‚Äù)
    • <thead> contains header row with <tr> and <th>
    • <tbody> holds data rows (<td>), with examples of colspan and rowspan
    • <tfoot> provides a footer row for summary or footer data
  • Table Cell Spanning:
    • colspan="2" merges two columns in one cell
    • rowspan="2" merges two rows in one cell
  • Unordered Lists (<ul>):
    • Default bullet style: <ul> ‚Ä¶ </ul>
    • Square bullets: <ul type="square">
    • Circle bullets: <ul type="circle">
  • Ordered Lists (<ol>):
    • Default numeric: <ol> ‚Ä¶ </ol>
    • Lowercase roman: type="i"
    • Uppercase roman: type="I"
    • Lowercase alpha: type="a"
    • Uppercase alpha: type="A"
  • Definition List (<dl>):
    • <dl> wraps the list
    • <dt> defines the term
    • <dd> provides the description
  • Anchor Wrapping an Image: Use <a href="... " target="_blank"> around an <img> to make the image clickable.
  • CSS for Tables:

    border: 2px solid yellowgreen;
