  • ID Selector (#):
    • Unique identifier for a single element on the page.
    • Syntax in HTML: <div id="firstdiv">‚Ä¶</div>
    • Syntax in CSS: #firstdiv { background-color: antiquewhite; }
  • Class Selector (.):
    • Can be applied to multiple elements.
    • Syntax in HTML: <div class="red">‚Ä¶</div>
    • Syntax in CSS: .red { color: red; border: 5px solid purple; }
  • Multiple Classes:
    • Assign more than one class by separating with spaces: <div id="thirddiv" class="black bg-yellow">‚Ä¶</div>
    • Here, .black and .bg-yellow both apply to the same element.
  • CSS Targeting:
    • To style by ID, prefix with # (e.g., #seconddiv { background-color: green; }).
    • To style by class, prefix with . (e.g., .bg-green { background-color: aqua; }).
  • HTML Anchor to ID:
    • You can link to a specific element by its ID: <a href="#firstdiv">Go to First Div</a>.
    • When clicked, the page scrolls directly to the element with id="firstdiv".
  • Key Takeaways:
    • Use IDs when you need to uniquely identify one element.
    • Use classes when you want to share styles across multiple elements.
