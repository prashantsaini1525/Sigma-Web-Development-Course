    • RGB (e.g., rgb(255, 0, 0))
    • RGBA (e.g., rgba(255, 0, 0, 0.5) ‚Äì alpha is opacity)

    • HSLA (e.g., hsla(0, 100%, 50%, 0.5))
    • CMYK (e.g., cmyk(0, 100%, 100%, 0) ‚Äì though less common in CSS)
    • Named colors (e.g., red, blue, green, white, black, yellow, purple)
  • Popular Named Colors:
    • Red: #FF0000
    • Green: #00FF00
    • Blue: #0000FF
    • Black: #000000
    • White: #FFFFFF

  • CSS Highlights (style.css in colors folder):
    • Reset (* { margin:0; padding:0; })
    • #colors gets a border: 4px solid blueviolet; and full‚Äêheight
    • Headings inside #colors are centered with backgrounds
    • Each <div> under #colors is 50vh tall, light‚Äêblue background, and black border
