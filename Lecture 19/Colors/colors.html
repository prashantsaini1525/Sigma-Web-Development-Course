Lets learn about colours.

Color can be represented in:

  1. Hexadecimal (like #FF0000)
  2. RGB (like rgb(255, 0, 0))
  3. RGBA (like rgba(255, 0, 0, 0.5))0.5 is opacity of the Color
  4. HSL (like hsl(0, 100%, 50%))
  5. HSLA (like hsla(0, 100%, 50%, 0.5))0.5 is opacity of the Color
  6. CMYK (like cmyk(0, 100%, 100%, 0))
  7. Named colors (like red, blue, green, white, black, yellow, etc.)

Popular color names:

  1. Red: #FF0000
  2. Green: #00FF00
  3. Blue: #0000FF
  4. Black: #000000
  5. White: #FFFFFF
  6. Yellow: #FFFF00
  7. Purple: #800080

Some other colors:

  1. Orange: #FFA500
  2. Gray: #808080
  3. Maroon: #800000
  4. Teal: #008080
  5. Indigo: #4B0082
  6. Aqua: #00FFFF
  7. Pink: #FFC0CB
