üìí Notes for Lecture 22: CSS Sizing Units ‚Äì px, rem, em, vh, vw, %, vmin

  • Pixels (px): Fixed size unit. Doesn't scale with screen. Example: width: 800px;

    Hinglish: Ye ek fixed value hai, screen chhoti ho ya badi ‚Äì element ka size same rahega.

  • Viewport Width/Height (vw, vh): Relative to the size of the browser window. 100vw means 100% of viewport width. 100vh means 100% of height.

    Hinglish: vw/vh browser window ke hisaab se size adjust karte hain.

  • em unit: Relative to the font-size of the parent. If parent is 16px, then 2em = 32px. 
    1em = 16px
2em = 32px
0.5em = 8px

    Hinglish: Ye parent font-size ka multiply hota hai.

  • rem unit: Relative to the html root font-size. If html { font-size: 12px } then 2rem = 24px everywhere. 
    1rem = 12px
2rem = 24px
3rem = 36px

    Hinglish: Ye root element se fix hota hai. Poore document me same rahega.

  • vmin unit: Minimum of vw and vh. 100vmin means 100% of smaller screen dimension.

    Hinglish: Jo screen ka chhota side hoga (width ya height), uska percent leta hai.

  • % unit: Relative to parent element. Example: width: 80%; will take 80% of parent width.

    Hinglish: Parent ke size ka kitna percent lena hai, ye decide karta hai.
